# Introduction
This is a simple workout timer and log application.  It is still in progress
Technologies:
..*C#
..*WebAPI
..*EF (Entity Framework) - In Memory Database
..*Bootstrap
..*javascript (ECMAScript/ES8)
..*jQuery
..*json
..*3rd Party jQuery libraries managed with Bower
..*CSS
..*HTML5
..*jsrender templating
